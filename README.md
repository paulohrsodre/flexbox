# Posicionando elementos com Flexbox em CSS

## Conteúdo do Curso

### Fundamentos
- Estrutura básica do display:flex;
  - Flex direction
  - Flex wrap
  - Flex flow
  - Justify content
  - Align items
  - Align content
  
- Estrutura básica do flex grow
  - Flex basis
  - Flex shrink
  - Flex
  - Order
  - Align self
 
## Projeto
- Landing page utilizando os fundamentos acima listados, responsividade e boas práticas
